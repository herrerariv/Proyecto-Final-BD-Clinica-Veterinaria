# Proyecto Final - Base de Datos Clínica Veterinaria

Este repositorio contiene el proyecto final de la materia Bases de Datos, basado en el escenario de una clínica veterinaria. El objetivo del proyecto es diseñar e implementar una base de datos relacional en PostgreSQL para gestionar propietarios, mascotas, veterinarios, citas, diagnósticos, tratamientos, medicamentos, procedimientos y facturación.

## Contenido del repositorio

* `documento/`: contiene el documento técnico del proyecto en formato PDF.
* `scripts/`: contiene los scripts SQL utilizados para crear la base de datos, insertar datos de prueba, realización de consultas y desarrollo de la programación en la base de datos.
* `diagramas/`: contiene el modelo Entidad-Relación (ER) en notación Chen y el modelo relacional (MR).

## Scripts SQL

Los scripts se encuentran organizados de la siguiente manera:

1. `veterinaria(DDL).sql`: creación de la base de datos, tablas y restricciones.
2. `veterinaria(DML).sql`: inserción de datos representativos.
3. `veterinaria(consultas).sql`: consultas SQL desarrolladas para responder a necesidades del negocio.
4. `veterinaria(programacion).sql`: funciones, procedimientos almacenados, transacciones y triggers.
